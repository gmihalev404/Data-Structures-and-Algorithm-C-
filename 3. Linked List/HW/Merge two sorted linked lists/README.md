# Merge two sorted linked lists

Този документ съдържа описанието на две задачи за работа със свързани списъци (Linked Lists).

---

## 1. Insert a Node at a Specific Position in a Linked List

Даден е указател към началния възел (**head**) на свързан списък и цяло число, което трябва да се вмъкне на определена позиция. Създайте нов възел с даденото число като негов `data` атрибут, вмъкнете този възел на желаната позиция и върнете началния възел (head) на списъка.

Позиция **0** указва началото на списъка, позиция **1** указва един възел след началото и т.н. Подаденият указател към началото може да бъде `null`, което означава, че първоначалният списък е празен.

### Example
* **head** сочи към: $1 \to 2 \to 3$
* **data** = $4$
* **position** = $2$

Новият списък е: $1 \to 2 \to 4 \to 3$



### Function Description
Завършете функцията `insertNodeAtPosition` със следните параметри:

* **SinglyLinkedListNode pointer llist**: референция към началото на списъка.
* **data**: целочислена стойност за вмъкване като `data` в новия възел.
* **position**: целочислена позиция за вмъкване на новия възел (индексация от нула).

**Returns**
* **SinglyLinkedListNode pointer**: референция към началния възел на обновения списък.

### Input Format
* Първият ред съдържа цяло число $n$ — броят на елементите в свързания списък.
* Всеки от следващите $n$ реда съдържа цяло число `SinglyLinkedListNode[i].data`.
* Следващият ред съдържа цяло число `data` — стойността на възела, който трябва да се вмъкне.
* Последният ред съдържа цяло число `position`.

### Constraints
* $1 \le n \le 1000$
* $1 \le SinglyLinkedListNode[i].data \le 1000$
* $0 \le position \le n$

### Sample Input
```text
3
16
13
7
1
2
```

### Sample Output
```text
1 2 3 3 4
```
